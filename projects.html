PC Assembler

Computer Science A Level coursework to help select the correct and compatible parts for a computer build.

Technologies

JavaScript

HTML

Java

CSS

Timeframe

Oct 2020 - Jan 2021

Status

Complete

PC Assembler was my first full project. It was a requirement for the OCR A Level Computer Science course to create a website. PC Assembler went through multiple iterations, as well as planning to make sure the project was completed by the strict deadline, which was 4pm on the 21st January 2021.

The Full write up for the project including design and development can be found here.

PC Assembler was one of 3 ideas I had which had to be approved by the teacher. Other ideas included a static HTML page using the Plex API to display media from a server for someone who does not have access to the server.

Pc Assembler was my first project which involved the development of custom API's in Java, as well as, creating a Web server for the project. This project can be described as full stack. Both the Front-end HTML, CSS and Javascript, as well as the back-end Java, API's and SQL database was programmed and designed by myself.

Due to the environment the project was created in, there is currently no live demo.
